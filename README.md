🚀 Gen AI LinkedIn Post Generator
An AI-powered application that automatically generates engaging and personalized LinkedIn posts using Generative AI. This project leverages prompt engineering and NLP techniques to create professional content for different topics, tones, and lengths.

📌 Features
✨ Generate high-quality LinkedIn posts instantly
🧠 Uses Generative AI for context-aware content
🎯 Supports different post lengths (Short, Medium, Long)
🌐 Multi-language support
🏷️ Tag-based content generation
⚡ Simple and easy-to-use interface

🛠️ Tech Stack
Programming Language: Python

Libraries/Tools:
Groq API (LLM integration)

JSON handling
Prompt Engineering
Environment: PyCharm

📂 Project Structure
project-genai-post-generator/
│── preprocess.py           # Data preprocessing
│── llm_helper.py           # LLM API integration
│── few_shot.py             # Few-shot examples for better prompts
│── main.py                 # Main script to generate posts
│── processed_posts.json    # Output generated posts
│── README.md               # Project documentation

⚙️ Installation
Clone the repository:
git clone https://github.com/Krupa-P-08/genai-linkedin-post-generator.git
cd genai-linkedin-post-generator

Create a virtual environment:
python -m venv venv
venv\Scripts\activate   # Windows

Install dependencies:
pip install -r requirements.txt

Set up your API key:
export API_KEY=your_api_key_here

▶️ Usage
Run the main script to generate posts:
python main.py

You can customize:
Post length (Short / Medium / Long)
Language
Topic/Tag
